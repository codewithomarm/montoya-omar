¿Qué es JavaScript y en qué se asemeja a Java

Javascript es un lenguaje de programación que permite agregar funcionalidades avanzadas a una página web. Le da la capacidad a una página web de ser dinámica e interactiva en vez de estática. Permite mostrar animaciones, mapas interactivos, cambiar fuentes, interactuar con el servidor, entre otras. Javascript se asemeja a Java ya que ambos son lenguajes de programación orientado a objetos. Ambos emplean encapsulación, abstracción, polimorfismo y herencia.
